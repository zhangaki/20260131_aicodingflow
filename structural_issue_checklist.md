# Structural Issue Checklist

1. Heading Hierarchy: Check if the headings are in the correct order (H1, H2, H3, etc.). Ensure there are no skips in the hierarchy (e.g., H1 followed by H3).
2. Orphan Links: Identify any links that are broken or lead to non-existent pages.
3. Anchor Map: Check if the anchor links within the pages are working correctly and pointing to the right sections.
4. Image Alt Text: Ensure all images have descriptive alt text for accessibility.
5. Mobile Responsiveness: Check if the website is responsive on different screen sizes.
